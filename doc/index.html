by Jerome Henin, Olaf Lenz, Cameron Mura and Jan Saam
This plugin provides procedures to handle periodic boundary conditions, i.e. to set and get the unitcell parameters, to wrap atoms into the central image, to unwrap atoms when they have been wrapped, and to draw the unit cell vectors. All of the procedures are able to handle non-orthorhombic periodic boundary conditions and changes of the unitcell parameters over time (as for example in constant pressure simulations). The PBCTools plugin and its documentation is hosted here.
